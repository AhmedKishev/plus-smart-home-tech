# Телеметрическая платформа для умного дома

# Описание проекта
Платформа для сбора, обработки и анализа телеметрии от датчиков умного дома. Система автоматически анализирует данные с датчиков и запускает соответствующие сценарии умного дома (например, включение света при обнаружении движения в тёмной комнате).
